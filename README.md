# Willow Wind Warbler

**The Willow Wind Warbler is your one-stop nest for all the family chirps, tweets, and tales. Get the scoop before it flies the coop!**

---

## About The Project

Welcome to the official repository for the Willow Wind Warbler family newsletter application. This project is a modern, web-based platform designed to digitally host, archive, and manage family newsletters.

Built with a modern tech stack, it provides a seamless experience for both readers and administrators.

**Core Features:**
*   **Latest Edition Viewer:** Displays the most recent newsletter in an easy-to-read PDF format.
*   **Historical Archive:** Browse and access all past editions, neatly organized by year.
*   **Secure Admin Panel:** A passwordless, email-based authentication system protects the administration area where newsletters are managed.
*   **Simple Upload/Delete:** Administrators can easily upload new editions or remove old ones from Firebase Cloud Storage.

### Built With

*   [Next.js](https://nextjs.org/)
*   [React](https://reactjs.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [ShadCN UI](https://ui.shadcn.com/)
*   [Firebase](https://firebase.google.com/) (Authentication & Cloud Storage)
